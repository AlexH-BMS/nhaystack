NHaystack

NHaystack is an open-source Niagara AX module, hosted at https://bitbucket.org/jasondbriggs/nhaystack, that serves up Project Haystack data directly from a Niagara AX station, via a RESTful protocol.

The development of NHaystack has been funded by J2 Innovations. NHaystack is licensed under the Academic Free License ("AFL") v. 3.0.

Status

NHaystack is currently in beta. This means that the project is mostly complete, but there may be bugs. As such, it is not ready yet for deployment in a production environment.

Getting Started

To get started, install nhaystack.jar into a station. Open the nhaystack palette, and drag-and-drop the NHaystackService onto your /Services folder.

This is all you need to do to get rolling. Your station is now automatically serving up all the control:ControlPoint objects and Histories as haystack 'point' recs, via the Haystack REST Api. All of the tags that are defined as being associated with points, like 'kind', 'unit', 'tz', 'his', 'cur' etc are automatically generated for you.
